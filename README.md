# XP Tracker — PWA

## Что это
Трекер задач с системой опыта. Нажимаешь задачу — получаешь XP. Бар заполняется, уровень растёт.
Работает офлайн, сохраняет прогресс, присылает ежедневные уведомления в шторку.

---

## Деплой на GitHub Pages (бесплатно, 5 минут)

### 1. Создай аккаунт на GitHub
https://github.com — если ещё нет.

### 2. Создай новый репозиторий
- Нажми "New repository"
- Назови: `xp-tracker`
- Тип: Public
- Нажми "Create repository"

### 3. Загрузи файлы
На странице репозитория нажми "uploading an existing file" и загрузи:
- index.html
- manifest.json
- sw.js
- icon-192.png  ← нужно создать (см. ниже)
- icon-512.png  ← нужно создать (см. ниже)

### 4. Включи GitHub Pages
- Settings → Pages
- Source: "Deploy from a branch"
- Branch: main / (root)
- Save

Через 1-2 минуты сайт будет на: `https://ВАШ_ЛОГИН.github.io/xp-tracker/`

---

## Иконки (нужны для PWA)

Самый простой способ — сгенерировать на https://favicon.io/favicon-generator/
- Text: XP
- Background: #c8f04a
- Font color: #0f0f0f
- Скачай, переименуй файлы в icon-192.png и icon-512.png

Или используй любую картинку 512×512 px.

---

## Установка на телефон (Android)

1. Открой `https://ВАШ_ЛОГИН.github.io/xp-tracker/` в Chrome
2. Появится баннер "Добавить на главный экран" — или через меню (⋮) → "Добавить на главный экран"
3. Готово — иконка появится как обычное приложение

При первом открытии разреши уведомления — будешь получать напоминания утром и вечером.

---

## Уведомления
- Утром (~9:00): напоминание начать зарабатывать XP
- Вечером (~20:00): сколько осталось до следующего уровня
- При повышении уровня — мгновенное уведомление

Уведомления работают только если хотя бы иногда открываешь приложение (Service Worker активен).
